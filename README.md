# ProjectMMO
 It is an MMO video game engine in the browser development in Javascript and nodeJS

# Install

`$ npm install`

# You can run

`$ npm start`

for default the application run in [http://localhost:3000/](http://localhost:3000/)

# Run test

`$ npm test`
